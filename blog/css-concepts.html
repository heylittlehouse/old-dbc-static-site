CSS Concepts

Question: What is the difference between displaying html elements inline and displaying them in an inline-block?

If, for example, we are modifying some divs with inline, inline will make the divs show up in a row, next to each other, from left to right. The divs will follow the size of their contents and you cannot modify their width and height or modify top and bottom margins and padding (but you can modify left and right margins and padding). With inline block on the other hand, the divs will also appear in row next to each other, but you can set the div's width and height and modify top and bottom margins and padding as well. Inline-block therefore combines the aspects of inline element and block element, which would force a break after each block element. You can see the differences between the three in the picture below which I got from here.

So if these were the only divs on the page, the CSS might look like this:

div {
background-color: red;
border: 2px solid black;
display: inline; (or display: inline-block, etc)
}
